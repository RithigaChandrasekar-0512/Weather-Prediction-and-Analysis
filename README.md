# Weather Data Analysis and Temperature Prediction

## Project Overview

This project analyzes historical Seattle weather data and predicts future maximum temperatures using Machine Learning techniques.

The dataset contains daily weather records including:

- Date
- Precipitation
- Maximum Temperature
- Minimum Temperature
- Wind Speed
- Weather Condition

## Objectives

- Perform data preprocessing
- Analyze weather trends
- Visualize temperature distributions
- Build a machine learning model
- Predict future temperatures
- Evaluate model performance

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset Features

| Feature | Description |
|----------|------------|
| date | Observation date |
| precipitation | Rainfall amount |
| temp_max | Maximum temperature |
| temp_min | Minimum temperature |
| wind | Wind speed |
| weather | Weather condition |

## Machine Learning Model

Linear Regression

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Workflow

1. Load Dataset
2. Data Cleaning
3. Feature Engineering
4. Exploratory Data Analysis
5. Model Training
6. Prediction
7. Evaluation

## Results

The model predicts future maximum temperatures using historical weather data and weather-related features.
